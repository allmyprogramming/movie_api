MyMovieDATA API Documentation

This API provides access to users and movies in the MyMovieDATA application. It includes routes for registering users, updating user data, managing favorite movies, and fetching movie data by title, genre, and director.

Base URL

http://localhost:8080

User Routes

1. Register a New User

POST /users

Request Body:

{
  "username": "string",
  "email": "string",
  "birthday": "YYYY-MM-DD"
}

Success Response:

{
  "message": "User registered successfully",
  "user": {
    "_id": "string",
    "Username": "string",
    "Email": "string",
    "Birthday": "YYYY-MM-DD"
  }
}

Error Response:

{
  "message": "Username and email are required"
}

2. Get All Users

GET /users

Success Response:

[
  {
    "_id": "string",
    "Username": "string",
    "Email": "string",
    "Birthday": "YYYY-MM-DD",
    "FavoriteMovie": "string"
  }
]

3. Get User by Username

GET /users/:username

Path Parameters:

:username

Success Response:

{
  "_id": "string",
  "Username": "string",
  "Email": "string",
  "Birthday": "YYYY-MM-DD",
  "FavoriteMovie": "string"
}

Error Response:

{
  "message": "No such user"
}

4. Update User Info

PUT /users/:username/update

Path Parameters:

:username

Request Body:

{
  "newUsername": "string",
  "email": "string",
  "birthday": "YYYY-MM-DD"
}

Success Response:

{
  "message": "User info updated",
  "user": {
    "_id": "string",
    "Username": "string",
    "Email": "string",
    "Birthday": "YYYY-MM-DD"
  }
}

Error Response:

{
  "message": "User not found"
}

5. Update User's Favorite Movie

PUT /users/:username/favoriteMovie

Path Parameters:

:username

Request Body:

{
  "favoriteMovie": "string"
}

Success Response:

{
  "message": "Favorite movie updated successfully",
  "user": {
    "_id": "string",
    "Username": "string",
    "Email": "string",
    "FavoriteMovie": "string",
    "Birthday": "YYYY-MM-DD"
  }
}

Error Response:

{
  "message": "User not found"
}

6. Remove a Favorite Movie

PUT /users/:username/favorites/remove

Path Parameters:

:username

Request Body:

{
  "movieId": "string"
}

Success Response:

{
  "message": "Movie removed from favorites",
  "user": {
    "_id": "string",
    "Username": "string",
    "Email": "string",
    "FavoriteMovie": "string",
    "Birthday": "YYYY-MM-DD"
  }
}

Error Response:

{
  "message": "User not found"
}
