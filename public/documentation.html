MyMovieDATA API Documentation

This API provides access to users and movies in the MyMovieDATA application. It includes routes for registering users, updating user data, managing favorite movies, and fetching movie data by title, genre, and director.

  • Register a new user
  • Get user details
  • Update user information
  • Manage a user's favorite movies (add, remove, view)

User Routes

1. Register a New User

Endpoint: POST /users

Registers a new user with a username, email, and optional birthday.

Request Body:

{
    "username": "exampleUsername",
    "email": "example@example.com",
    "birthday": "1992-11-05"
}

Response:

{
    "message": "User registered successfully",
    "user": {
        "Username": "exampleUsername",
        "Email": "example@example.com",
        "Birthday": "1992-11-05T00:00:00.000Z"
    }
}

2. Get User by Username

Endpoint: GET /users/:username

Retrieve a user's details by their username.

Response:

{
    "Username": "exampleUsername",
    "Email": "example@example.com",
    "FavoriteMovie": ["Interstellar", "Inception"],
    "Birthday": "1992-11-05T00:00:00.000Z"
}

3. Update User Info

Endpoint: PUT /users/:username/update

Update a user's information (username, email, or birthday).

Request Body:

{
    "newUsername": "newExampleUsername",
    "email": "newemail@example.com",
    "birthday": "1990-10-20"
}

Response:

{
    "message": "User info updated",
    "user": {
        "Username": "newExampleUsername",
        "Email": "newemail@example.com",
        "Birthday": "1990-10-20T00:00:00.000Z"
    }
}

4. Add Movie to Favorite Movies

Endpoint: POST /users/:username/favorites

Add a movie to the user's list of favorite movies.

Request Body:

{
    "favoriteMovie": "Pulp Fiction"
}

Response:

{
    "Username": "exampleUsername",
    "Email": "example@example.com",
    "FavoriteMovie": ["Interstellar", "Inception", "Pulp Fiction"],
    "Birthday": "1992-11-05T00:00:00.000Z"
}

5. Remove Movie from Favorite Movies

Endpoint: DELETE /users/:username/favorites

Remove a movie from the user's list of favorite movies using the movie title.

Request Body:

{
    "favoriteMovie": "Pulp Fiction"
}

Response:

{
    "Username": "exampleUsername",
    "Email": "example@example.com",
    "FavoriteMovie": ["Interstellar", "Inception"],
    "Birthday": "1992-11-05T00:00:00.000Z"
}

6. Get User's Favorite Movies

Endpoint: GET /users/:username/favorites

Retrieve a user's list of favorite movies.

Response:

[
    "Interstellar",
    "Inception"
]

Summary

  • POST /users: Register a new user
  • GET /users/:username: Get user details
  • PUT /users/:username/update: Update user information
  • POST /users/:username/favorites: Add a movie to the user's favorites
  • DELETE /users/:username/favorites: Remove a movie from the user's favorites
  • GET /users/:username/favorites: Get user's list of favorite movies
