Movie API Documentation

Base URL: http://localhost:8080

Endpoints

1. Get All Movies

Method: GET

URL: /movies

Description: Returns a list of all movies in the database.

2. Get Movie by Title

Method: GET

URL: /movies/:title

Description: Returns details of a specific movie by its title (e.g., "Inception").

Response: 

{
    "Title": "Inception",
    "Description": "A thief who enters the dreams of others to steal secrets from their subconscious.",
    "Genre": {
        "Name": "Sci-Fi",
        "Description": "Science fiction movies explore futuristic concepts."
    },
    "Director": {
        "Name": "Christopher Nolan",
        "Bio": "British-American filmmaker known for his mind-bending films and time manipulation.",
        "birthYear": 1970
    },
    "Actors": ["Leonardo DiCaprio", "Joseph Gordon-Levitt", "Elliot Page"],
    "ImageUrl": "https://image.url/inception.jpg",
    "Featured": true
}

3. Get Genre Description

Method: GET

URL: /genres/:name

Description: Returns the description of a movie genre (e.g., "Sci-Fi").

Response: 

{
    "Description": "Science fiction movies explore futuristic concepts."
}

4. Get Director Info

Method: GET

URL: /directors/:name

Description: Returns the bio and birth year (and optionally death year) of a director by their name (e.g., "Christopher Nolan").

Response: 

{
    "Bio": "British-American filmmaker known for his mind-bending films and time manipulation.",
    "BirthYear": 1970
}

5. Register a New User

Method: POST

URL: /users

Body:

{
    "Name": "John Doe",
    "Email": "johndoe@example.com",
    "FavoriteMovie": "Inception",
    "Birthday": "1990-01-01",
    "IsActive": true
}

Description: Registers a new user with their name, email, favorite movie, birthday, and active status.

Response: 

{
    "message": "User registered successfully",
    "user": {
        "Name": "John Doe",
        "Email": "johndoe@example.com",
        "FavoriteMovie": "Inception",
        "Birthday": "1990-01-01T00:00:00.000Z",
        "IsActive": true
    }
}

6. Update User Info (Email, Favorite Movie, etc.)

Method: PUT

URL: /users/:email

Body:

{
    "Email": "alice.green@example.com",
    "FavoriteMovie": "The Matrix"
}

Description: Updates a user's email and favorite movie.

7. Add Movie to Favorites

Method: POST

URL: /users/:email/favorites/:movieTitle

Description: Adds a movie to the user's favorite movie list.

8. Remove Movie from Favorites

Method: DELETE

URL: /users/:email/favorites/:movieTitle

Description: Removes a movie from the user's favorite movie list.

9. Delete User

Method: DELETE

URL: /users/:email

Description: Deletes a user by their email address.
