Movie API Documentation

Base URL: http://localhost:8080

Endpoints

1. Get All Movies

Method: GET

URL: /movies

Description: Returns a list of all movies.

2. Get Movie by Title

Method: GET

URL: /movies/:title

Description: Returns details of a specific movie.

3. Get Genre Description

Method: GET

URL: /movies/genre/:genreName

Description: Returns the description of a movie genre.

4. Register a New User

Method: POST

URL: /users

Body:

{
    "username": "john_doe",
    "email": "johndoe@example.com"
}

Description: Registers a new user.

5. Update Username

Method: PUT

URL: /users/:username

Body:

{
    "newUsername": "johnny_doe"
}

Description: Updates a user's username.

6. Add Favorite Movie

Method: POST

URL: /users/:username/favorites/:movieTitle

Description: Adds a movie to the user's favorites.

7. Remove Favorite Movie

Method: DELETE

URL: /users/:username/favorites/:movieTitle

Description: Removes a movie from the user's favorites.

8. Delete User

Method: DELETE

URL: /users/:username

Description: Deletes a user account.
